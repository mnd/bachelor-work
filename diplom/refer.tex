Меринов~Н.Д. Динамическая диспетчеризация в языке Haskell, Квалификационная работа на степень бакалавра наук: стр.50, библ. 11 назв., приложений 4.
\vspace{0.3cm}

Ключевые слова: ДИНАМИЧЕСКАЯ ДИСПЕТЧЕРИЗАЦИЯ, HASKELL.
\vspace{0.3cm}

В тексте описана реализация динамической диспетчеризации в языке
Haskell на примере проблемы интерактивного взаимодействия с системой
работающей с данными большого числа типов. Рассмотрены существующие
решения для упрощения управления данными разных типов. В конце
предложено решение автоматического построения целевой системы на
основе динамического типа с нарушением безопасности языка.
