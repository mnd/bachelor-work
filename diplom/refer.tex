Меринов~Н.Д. Динамический полиморфизм в языке Haskell, дипломная работа: стр.48, библ. 11 назв., приложений 4.
\vspace{0.3cm}

Ключевые слова: ПОЛИМОРФИЗМ, ТЕОРИЯ ТИПОВ, HASKELL.
\vspace{0.3cm}

В тексте описана проблема интерактивного взаимодействия с системой
работающей с данными большого числа типов. Рассмотрены существующие
решения для упрощения управления данными разных типов. В конце
предложено решение автоматического построения целевой системы на
основе динамического типа с нарушением безопасности языка.
